Lesson-4

FONT: Used for defining font information. OBJECT: Used for embedding media files into the page. INPUT: For inputs, such as a form input or radio button. IMG: Images used throughout the page. TABLE: Allows for a tabular, data view of data. INNERTEXT: allows you to add alternative text content to an image element. SPAN: Allows bold, italic and highlighted text in some cases.

APBODY: A way of defining an alternate section of content for people using assistants, such as a screen reader. FRAMESET: A way to create multiple frames that use a common frame. SECTION: A more efficient section method compared to div, used for defining sections in a page. INLINE: Allows you to embed a JS code within the HTML.
