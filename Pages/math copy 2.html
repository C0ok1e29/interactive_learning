Lesson 1

Example: Write down the place value of each digit in the number 543. The correct place value of each digit in the number can be expressed as follows: 5 × 100 = 500 or 5 hundreds 4 × 10 = 40 or 4 tens 3 × 1 = 3 or 3 ones The place value is the position of each digit in a number. The place value of digits is determined as ones, tens, hundreds, thousands,ten-thousands and so on, based on their position in the number. For example, the place value of 1 in 1002 is thousands, i.e.1000.
