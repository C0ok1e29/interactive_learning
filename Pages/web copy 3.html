Lesson

FRAME: A frame is a way of defining a certain section, and then loading a page within the frame. EM: Defines a section of text as a specific emphasis, such as bold, italic etc.. COLSPAN: Used to span a number of columns in a table. PADDING: Allows you to use CSS padding to extend the bounds of an element. BODRER-image: Allows you to put an image as a border.

HEAD: Used to define meta data for the page, for example keywords and description and information. TITLE: Used for the page title. META: Used for meta data for the page, where you can add meta tags which provide information to search engines about the page, for better ranking and visibility on search results. BODY: The main container for the page. SCRIPT: Allows you to integrate browser Javascript code onto the page. LINK: Used for defining external external for other pages.
